# PoDMaster

This repository serves as a centralized collection of notes, code, and materials for various master's level courses. Each top-level directory corresponds to a specific course.

## Folder Structure

The repository is organized by course, with each course having its own dedicated directory. The general structure for each course folder is as follows:

```
COURSE_NAME/
├── class_notes/
├── code/
├── final_project/
└── professor_notes/
```

- **`class_notes/`**: Contains personal notes taken during lectures.
- **`code/`**: Holds programming assignments, labs, and code examples.
- **`final_project/`**: Dedicated to the final project materials for the course.
- **`professor_notes/`**: Contains lecture slides, PDFs, and other materials provided by the professor.

## Courses

- **`ASPA/`**: Advanced Studies in Public Administration
- **`LCP/`**: Laboratory of Computational Physics
- **`MAPD/`**: Methodologies for Advanced Programming and Design
- **`ML/`**: Machine Learning
- **`MTP/`**: Models of Theoretical Physics
- **`QIAP/`**: Quantum Information and Photonics
